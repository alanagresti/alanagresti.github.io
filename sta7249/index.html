STA 7249: GENERALIZED LINEAR MODELS

Links about generalized linear models

The web site for Dr. Pat Altham at Cambridge University in the UK has a very nice set on notes about GLMs. Her site is also a good source of examples of the use of Splus and R.

A website I've created for my book Categorical Data Analysis (2nd edition, Wiley 2002) has details about various software for GLMs, with main emphasis on methods for conducting categorical data analysis. Dr. Brett Presnell has a useful set of links for the R software, which is free.
