DATA FILES for "STATISTICAL METHODS FOR THE SOCIAL SCIENCES,"

By Alan Agresti, Pearson, 5th edition, 2018

This is the website for the newest edition of "Statistical Methods for the Social Sciences," which was published in January 2017. For some powerpoint slides from a course I taught from this book for the Statistics Department at Harvard in 2010, go to data files. For data files for examples and exercises in the book, go to data files. These are text data files that can easily be inported for use with R or SAS. For data files in the form ready for use on Stata (version 14), go to Stata 14 data files. For data files in the form ready for use with version 13 of Stata, thanks to Prof. James Endersby, go to Stata 13 data files. For SPSS, go to SPSS data files.
